# frontend


views 存放页面

router 文件夹下的index.js文件存储路由信息

store 文件下的index.js文件存储静态变量信息

component 存放组件

## Project Setup

```sh
npm install
npm install axios

```

### Compile and Hot-Reload for Development

```sh
npm run dev
```

### Type-Check, Compile and Minify for Production

```sh
npm run build
```
